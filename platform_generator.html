🚀 Quick Start

  1. Download and extract the platform files to a directory
  2. Install dependencies: pip install -r requirements.txt
  3. Install Graphviz (for Petri net rendering): winget install Graphviz or download from graphviz.org
  4. Launch the dashboard: streamlit run app.py
  5. Upload a CSV or XES event log file
  6. Map columns (case_id, activity, timestamp) — auto-detected if standard names
  7. Select a process type from the sidebar (e.g., "Procure-to-Pay")
  8. Click "🚀 Analyze" — the platform will ingest, discover, diagnose, and generate an executive brief
